# Mi App - Catálogo con React y Vite

**Descripción**  
Aplicación web creada con **React + Vite** que incluye una página de presentación y un catálogo de productos conectado a una API/backend. Plantilla pensada para desarrollo local y despliegue en servicios como Vercel o Netlify.

**Requisitos**  
- Node.js 16 o superior  
- Git  
- (Opcional) Docker si vas a levantar la base de datos en contenedores

**Instalación y ejecución local**  
```bash
# clonar o crear el proyecto
git clone https://github.com/tu-usuario/mi-app.git
cd mi-app

# instalar dependencias
npm install

# servidor de desarrollo
npm run dev

# build para producción
npm run build

# previsualizar build localmente
npm run preview
